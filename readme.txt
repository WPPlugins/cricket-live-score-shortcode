== Description ==
You can display Live Cricket Scores & Results of latest matches using shortcodes. The score is fetched from [CricBuzz RSS feed](http://synd.cricbuzz.com/j2me/1.0/livematches.xml) and hence are always the latest.

<h2> For Cricket Live Score use a below shortcode </h2>
<code>[clsw_scores]</code>

Credits : <a href="http://synd.cricbuzz.com/j2me/1.0/livematches.xml" target="_blank">CricBuzz RSS feed</a>

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==
= How to Use ? =
For display Cricket Live Score use shortcode 
[clsw_scores]

For display Cricket Recent Match Results use shortcode 
[clsw_results]

For display Cricket Upcoming Matches list use shortcode 
[clsw_upcoming]

= Is there any widget available for live scores? =
No. This plugin contains 3 shortcodes for live scores, latest match results & upcoming matches. We are working on live score widgets.
